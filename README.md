# A3MN
Nature de la cellule  A3MN est à la fois :  *  un média stratégique ; *  un centre de réflexion et d’analyse ; *  un dispositif de formation et d’éducation numérique ; *  une plateforme de détection et de structuration de talents ; un outil de développement de projets et d’activités économiques ; un espace d’influence intellectuelle panafricaine.
